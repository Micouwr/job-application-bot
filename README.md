# Job Application Bot

Your AI-powered resume tailoring assistant for IT/DevOps/AI Governance roles.

## Features

- Desktop GUI Application
- AI-Powered Matching
- One-click tailored resume + cover letter
- SQLite Database
- Resume Management
- 100 percent ASCII Clean
- Standalone Build Ready

## Quick Start

1. Install dependencies:
   pip install -r requirements.txt

2. Set up environment:
   cp .env.example .env
   # Edit .env and add your GEMINI_API_KEY

3. Run:
   python gui/tkinter_app.py

4. Build:
   python build_standalone.py

## Requirements

- Python 3.9+
- Gemini API key
